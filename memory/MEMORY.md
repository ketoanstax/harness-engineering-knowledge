# Index Bộ nhớ Cộng tác (Collaborative Memory Index)

Tài liệu này lưu trữ và lập chỉ mục các file ghi nhớ lâu dài về thói quen làm việc, vai trò người dùng và bài học rút ra để đảm bảo AI Agent không bị mất bối cảnh qua các phiên làm việc.

## 🗂️ Danh sách File Bộ nhớ
- [user_profile.md](memory/user_profile.md) — Hồ sơ người dùng, sở thích lập trình và giao tiếp.
- [feedback_log.md](memory/feedback_log.md) — Nhật ký phản hồi, sửa lỗi và các quy tắc tự động hóa phát sinh.
- [project_context.md](memory/project_context.md) — Bối cảnh dự án, trạng thái hiện tại và lộ trình phát triển.
- [RULE.md](memory/RULE.md) — Quy tắc vận hành bộ nhớ và feedback loop tự động.
