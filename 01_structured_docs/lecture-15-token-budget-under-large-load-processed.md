# Bản chắt lọc tự động - Lecture 15

## 💡 Key Takeaways
- Token Budget của Agent dễ bị cạn kiệt khi tải lớn.
- Blast Radius Isolation lớp mạng lưới giúp bảo vệ ngân sách token.

## 🗝️ Keywords & Core Concepts
- **Blast Radius Isolation**: Cô lập cứng môi trường thực thi của Agent để bảo toàn hệ thống.
- **Token Load Control**: Kiểm soát tải token bằng cách giới hạn request API tối đa của Agent.

## 📝 AI-Ready Summary
Bài giảng phân tích cách quản trị token budget dưới tải lớn.
